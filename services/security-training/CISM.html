The Certified Information Security Manager

Course Over View:

The Certified Information Security Manager (CISM) certification is a globally recognized credential designed for professionals who manage, design, oversee, and assess an enterprise's information security. This comprehensive course is tailored to equip you with the necessary skills and knowledge to excel in information security management.

Who Should Attend?

  1. Information Security Managers
  2. Aspiring Information Security Managers
  3. IT Consultants
  4. IT Directors and Managers
  5. Security Auditors
  6. Compliance Officers
  7. Risk Managers

Course Objectives

  1. Develop and manage an enterprise information security program.
  2. Establish and maintain an information risk management framework.
  3. Design, implement, and manage security controls.
  4. Understand and comply with legal and regulatory requirements.

Course Modules

Information Security Governance

  • Establishing and maintaining an information security governance framework.
  • Aligning information security strategy with organizational goals.
  • Understanding roles, responsibilities, and accountability.

Information Risk Management

  • Identifying and assessing information security risks.
  • Implementing risk mitigation strategies.
  • Developing and maintaining an information security risk register.

Information Security Program Development and Management

  • Designing and implementing information security programs.
  • Integrating security into business processes.
  • Managing information security projects.

Information Security Incident Management

  • Developing an incident response plan.
  • Identifying and responding to information security incidents.
  • Conducting post-incident analysis and implementing improvements.
